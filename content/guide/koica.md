# KOICA ODA 전문가 신청 절차

## 1. 모집 공고 확인

KOICA 및 외교부 국제개발협력본부에서 연중 모집 공고를 게시합니다.

- KOICA 채용공고: https://www.koica.go.kr/career
- ODA Korea: https://www.odakorea.go.kr/job

## 2. 지원 단계

### STEP 1: 사전 준비
- 최신 이력서 (영문/국문)
- 자기소개서 (동기, 강점, 기여 방안)
- 학력/경력 증명서
- 어학 성적표 (TOEFL, IELTS, JLPT 등)
- 추천서 2부
- 여권 사본
- 관련 자격증 사본

### STEP 2: 온라인 지원
1. KOICA 채용 시스템 회원가입
2. 모집 공고 선택
3. 양식에 맞춰 정보 입력
4. 필요 서류 업로드
5. 지원서 최종 제출

### STEP 3: 서류 심사
- 제출된 서류 기반 자격 요건 검증
- KOICA 내부 심사 + 외부 전문가 평가
- 심사 결과 통보 (통상 2~4주)

### STEP 4: 면접 심사
- 1차: 실무 면접 (관련 분야 전문가)
- 2차: 인성 면접 (KOICA 임직원)
- 최종: 고위 임원 면접 (일부 사업)

### STEP 5: 신체 검사
- 해외 파견 가능 여부 확인
- 필수 예방접종 이력 확인

### STEP 6: 사전 교육
- KOICA 아카데미에서 4주 교육
- ODA 이해, 개발협력 이론, 문화 적응, 안전 교육

### STEP 7: 파견
- 현장 배치 및 출국
- 정기 모니터링 및 평가

## 3. 심사 기준

| 항목 | 배점 |
|------|------|
| 관련 경력 | 30% |
| 학력 및 자격 | 20% |
| 어학 능력 | 20% |
| 면접 평가 | 20% |
| 기타 (추천서, 봉사) | 10% |

## 4. 처리 기간

| 단계 | 소요 기간 |
|------|----------|
| 공고 ~ 마감 | 2~4주 |
| 서류 심사 | 2~3주 |
| 면접 심사 | 1~2주 |
| 신체 검사 | 1주 |
| 사전 교육 | 4주 |
| 파견 준비 | 2~4주 |
| 총 기간 | 약 3~4개월 |

## 5. 주의 사항

- **필수 확인 사항**
  - 여권 유효기간 (최소 6개월 이상 남아있어야 함)
  - 가족 동반 가능 여부 (사업별 상이)
  - 안전 등급 (해당 국가)
  - 보험 가입 의무

## 7. 자주 하는 질문

**Q. 동시에 여러 사업에 지원 가능한가요?**
네, 중복 지원 가능하지만 최종 합격 시 1개 사업만 선택해야 합니다.

**Q. 지원서 작성 시 팁이 있나요?**
구체적인 성과와 지표를 사용하여 본인의 기여도를 명확히 보여주세요. 개발도상국 관련 경험이 있다면 반드시 강조하세요.

**Q. 파견 중에는 한국과 연락이 되나요?**
네, 대부분 현지에서도 인터넷과 국제 전화가 가능합니다. 정례 보고 일정에 따라 정기적으로 소통합니다.
